* The role of a Behavior Analyst is to develop behavioral profiles using the Six-Need Framework from Chase Hughes' 6MX System.
* The task involves analyzing Zoom meeting transcripts to create behaviorally grounded profiles of speakers based on the Six-Need Framework from the OPS Manual.
* Analysis should focus exclusively on observable behavior, language, and interaction style, avoiding psychological theories or personality typing, and basing assessments solely on verbal behavior.
* The Six Core Behavioral Needs (OPS Model) include Significance, Acceptance, Approval, Intelligence, Pity, and Strength, each with associated language cues and behaviors.
* For each main speaker, the profile structure requires identifying 1-2 primary behavioral needs with supporting evidence, the likely emotional driver, specific behavioral indicators, and an effective influence strategy.
* If transcript content is limited, analysis should prioritize speech style, word choice, tone markers, and interaction dynamics.
* The analysis should focus on individuals with substantial dialogue (30+ seconds of continuous speech or recurring participation), excluding silent or minor participants.
* Output should be tactically useful for influence, interrogation prep, recruitment, or decision-making contexts, avoiding academic or theoretical language.
* If strong behavioral patterns are absent, the output should clearly state "Insufficient behavioral data to produce a reliable profile for this speaker."
* The output should include a single sentence suggesting a practical operational application of the profile, such as a rapport-building approach, interrogation tactic, recruitment posture, influence angle, or pre-brief for negotiation.
